
# TK-Shopping-Mall

**태광 쇼핑몰**을 참고하여 제작한 간단한 쇼핑몰 **토이 프로젝트**입니다.  
자바와 스프링 프레임워크를 기반으로 쇼핑몰의 기본 기능을 구현하였습니다.

---

## 💻 사용 기술

- **백엔드:** Java, Spring Framework (5.x), MyBatis, Oracle Database
- **프론트엔드:** HTML, CSS, JavaScript
